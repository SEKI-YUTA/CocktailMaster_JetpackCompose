## APIサーバーの用意
プロダクトフレーバーを"emulator"に設定した場合は、[ここから](https://github.com/SEKI-YUTA/Cocktail_API_New)API側のプログラムをダウンロードして用意してください
プロダクトフレーバーを"physical"に設定した場合は、別途用意する必要はありません。
